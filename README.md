# Student-Performance-Analysis

## Objective
To explore, clean, analyze, and visualize student performance data.

## Dataset
- student-mat.csv

## Steps Performed
- Data Exploration
- Data Cleaning
- Data Analysis
- Data Visualization

## Key Insights
- Average grade ≈ 10.4  
- Weak correlation between study time and grades  
- Female students slightly perform better  

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn
